# ENT Clinic Patient Portal

A modern full-stack web-application designed for ENT (ear, nose, throat) clinic.

## Features (MVP)

- **Authentication & authorization**
- **Role-based access**
- **Simple appointment booking flow(for now)**
- **User profile**
- **Protected routes**
- **Cache management**

## Tech stack

**Frontend**

- React 18 (Vite)
- React Router v6 (Routing & Layouts)
- TanStack Query v5 (Server state management & data fetching)
- Tailwind CSS (Utility-first styling)
- shadcn/ui (Customizable UI components)
- React Hot Toast (Notifications)
- Lucide React (Icons)

**Backend**

- Supabase (PostgreSQL Database, Authentication, Row Level Security)
